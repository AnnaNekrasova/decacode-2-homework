# Домашнее задание к уроку 4 – Классы в JavaScript #1

Написать приложение, которое:
1. Считывает `products.csv` файл;
2. Создаёт массив объектов с продуктами;
3. Переводит цену продукта в рублях в цену в долларах;
4. Записывает файл `result.json` с финальным массивом объектов;

Пример вызова приложения:
```bash
$ node index.js
```

Пример `products.csv` - файла можно найти [здесь](products.csv).
Пример `JSON`-файла можно найти [здесь](result.json).

---

### Подсказки

1. Сперва необходимо прочитать файл `products.csv` с помощью `fs.readFileSync` и записать его значение в переменную
2. Затем необходимо описать класс `Product` у которого 2 свойства: `title`, `price`; Причём свойство `price` – объект с
   двумя ключами: `rubles` и `dollars`.
3. После чего необходимо преобразовать контент файла `products.csv` в массив вида `[{title: 'Milk', price: 100}]`;
4. Пройтись по этому массиву с помощью функции `map`, преобразовав все объекты в элементы класса `Product`;
5. Записать файл на диск, используя `JSON.stringify` и `fs.writeFileSync`;
